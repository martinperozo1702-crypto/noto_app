# Noto 📝

App de gestión de tareas diarias construida desde cero con HTML, CSS y JavaScript.

## ¿Qué hace?

- Agregar, editar y eliminar tareas
- Marcar tareas como completadas
- Filtrar por pendientes, todas o completadas
- Barra de progreso del día en tiempo real
- Papelera con opción de restaurar tareas eliminadas
- Diseño responsivo adaptable a cualquier dispositivo
- Datos guardados en el navegador con localStorage

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript
